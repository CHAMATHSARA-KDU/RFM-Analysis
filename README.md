# RFM-Analysis
Project Overview
This project presents a Power BI dashboard designed to explore and visualize customer behavior using demographic insights and RFM (Recency, Frequency, Monetary) analysis.
The goal is to help businesses identify their most valuable customers, understand sales distribution patterns, and make data-driven marketing and retention decisions.
